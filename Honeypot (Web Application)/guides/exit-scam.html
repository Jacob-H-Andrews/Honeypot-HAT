What is an Exit Scam?

An exit scam is a trick of trust when an existing corporation ceases delivering orders while still collecting payment for new orders. If the entity has a decent reputation, it will take some time for orders not to be delivered to be generally accepted, so the entity will instead run off with the money received for unshipped orders.


Examples
- The darknet market (online black market) Evolution is cited as the biggest exit scam yet as of 2016. The administrators apparently made off with $12 million in bitcoin, which was held in escrow on the marketplace.
- The WSM (Wall Street Market) exit scam of 2019 had $14.2 million worth of cryptocurrencies stolen, just before the site was seized by the authorities.
